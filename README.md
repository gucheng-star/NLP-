
**项目名称：基于LSTM的微博舆情情感分析系统**

**项目简述**  
设计并实现了一个面向中文微博文本的情感分析系统，旨在对用户评论进行积极/消极情感分类。项目采用双向LSTM模型，结合腾讯预训练词向量，在12万条标注数据上完成训练与评估，并开发了图形化交互界面，支持实时情感预测。

**核心工作**  
- 对原始微博数据进行清洗、字级别分词、序列填充与截断，并映射为词向量输入；  
- 搭建基于双向LSTM的深度学习模型，引入早停机制防止过拟合，提升训练效率；  
- 在测试集上评估模型性能，计算准确率、精确率、召回率与F1分数，并绘制混淆矩阵；  
- 基于Tkinter开发桌面端交互界面，支持用户输入文本并实时返回情感分类结果。
